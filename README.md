# FNSUS-DOC

Repositório central da documentação do Sistema para Geração de Relatórios Situacionais dos Municípios Brasileiros (FN-SUS).

## Sobre

Este projeto reúne a documentação técnica e os principais resultados do FN-SUS, uma iniciativa para apoiar respostas rápidas a emergências em saúde pública, fortalecendo a atuação da Força Nacional do Sistema Único de Saúde.

## Documentação

A documentação é construída com [MkDocs Material](https://squidfunk.github.io/mkdocs-material/), proporcionando uma navegação moderna e responsiva.

## Como executar localmente

1. **Clone o repositório:**
    ```bash
    git clone https://github.com/FNSUS/FNSUS-DOC.git
    ```

2. **Acesse a pasta do projeto:**
    ```bash
    cd FNSUS-DOC
    ```

3. **Inicie o servidor de documentação:**
    ```bash
    make serve
    ```

4. **Acesse no navegador:**
    ```
    http://127.0.0.1:8000
    ```
